# 快捷键

| 快捷键         | 功能描述               |
|----------------|------------------------|
| Ctrl+S         | 保存项目               |
| Ctrl+Z         | 撤销                   |
| Ctrl+Y         | 重做                   |
| Ctrl+C/V/X     | 复制/粘贴/剪切         |
| Ctrl+T         | 物编搜索               |
| F5             | 启动地图测试           |
| ...            | ...                    |
