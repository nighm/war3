# 快捷键

## 常用操作
| 快捷键         | 功能描述               |
|----------------|------------------------|
| Ctrl+S         | 保存项目               |
| Ctrl+Z         | 撤销                   |
| Ctrl+Y         | 重做                   |
| Ctrl+C/V/X     | 复制/粘贴/剪切         |
| Ctrl+T         | 物编搜索               |
| F5             | 启动地图测试           |

## 地形与单位编辑
| 快捷键         | 功能描述               |
|----------------|------------------------|
| Shift+拖动     | 精细移动单位/装饰物    |
| Delete         | 删除选中对象           |
| Ctrl+鼠标滚轮  | 缩放视图               |

## 物编与脚本
| 快捷键         | 功能描述               |
|----------------|------------------------|
| Ctrl+F         | 查找物编字段           |
| Ctrl+/         | 注释/取消注释脚本      |

## 其他
| 快捷键         | 功能描述               |
|----------------|------------------------|
| Ctrl+Shift+P   | 打开命令面板           |
| Ctrl+Tab       | 切换标签页             |
| ...            | ...                    |
