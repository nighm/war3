# 新成员入门指引

欢迎加入Y3地图开发项目团队！请按照以下步骤快速上手：

## 1. 环境准备
- 安装Y3编辑器（如已安装，确认路径无误）
- 安装魔兽争霸3本体
- 克隆或下载本项目代码

## 2. 阅读文档
- 阅读 docs/README.md，了解文档体系和导航
- 重点阅读“教学文档”目录下的入门章节

## 3. 实践操作
- 按照“新建地图项目.md”完成第一个地图创建
- 尝试地形编辑、单位添加、物编修改等基础操作

## 4. 反馈与协作
- 将遇到的问题补充到FAQ或术语表
- 认领并完善某一部分文档内容
- 有任何疑问及时在团队群或Issue中反馈

祝你开发愉快，团队共同进步！
